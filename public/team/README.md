# Team Photos Directory

Place all your team members' photos in this folder. 

**Format Guidelines:**
- **Dimensions:** 800px (Width) x 1200px (Height) or 600x800.
- **Ratio:** Vertical Portrait (3:4 or 2:3 ratio).
- **Format:** `.jpg` (recommended for smaller file size) or `.png`.
- **File Size:** Under 500KB per image.

**How to use them:**
Once you drop a photo here (e.g., `alice.jpg`), you can reference it in your code like this:
`/team/alice.jpg`

Update the `app/team/page.tsx` file's `photoUrl` to match the file names you drop here.
